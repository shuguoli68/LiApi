# LiApi
### 说明：  
一个后台应用，只写api接口。  
由于各大数据平台不再提供免费的api接口，实行实名制或次数有限，故此自己编写api接口。
不仅可以练习编写Java后台程序，还可以在学习前端知识框架时给自己提供api接口 

### 开发环境、工具：  
IntelliJ IDEA 2019.2.1 x64  Community版  
SpringBoot + Mybatis,数据存储在UniServer，一个小的无需安装的数据库软件  
框架：Mybatis、pagehelper
